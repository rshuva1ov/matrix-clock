# Matrix Clock

Полноэкранные матричные часы на Tauri + Rust + React.

## Технологии

- Tauri 2 (Rust backend)
- React + TypeScript
- Vite
- SCSS-модули, FSD-архитектура, алиасы как в `assessment/frontend`

## Запуск

```bash
pnpm install # либо npm install / yarn / bun
pnpm dev     # или npm run dev
pnpm tauri:dev
```
